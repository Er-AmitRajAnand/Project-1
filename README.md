# Amit Raj Anand — Portfolio

Personal portfolio site, built on the [Logsfolio](https://themewagon.com/themes/logsfolio/) Next.js + Tailwind CSS template. Content is sourced from `public/data.json`.

## Getting Started

1. Install dependencies
```bash
npm install
```
2. Run the development server
```bash
npm run dev
```
